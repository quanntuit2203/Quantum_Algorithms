Commands Example Explain
reset num_qubits reset 2 Set up a quantum simulation, with num_qubits qubits.
write target_qubits write 3 Write the specified value to the target qubits.
not_gate target_qubits not 1 Perform a logical NOT operation on the qubits specified by target_qubits
cnot_gate target_qubits,condition_qubits cnot 3 0 Perform a logical CNOT operation on the qubits specified by target_qubits, using condition_qubits as the condition
hadamard target_qubits hadamard 3 Perform a HADAMARD operation on the qubits specified by target_qubits
chadamard target_qubits,condition_qubits chadamard 4 2 Perform a CHADAMARD operation on the qubits specified by target_qubits, using condition_qubits as the condition
swap qubit1,qubit2 swap 1 0 Swap two qubits specified by qubit1 and qubit2
phase theta_degrees, target_qubits phase 315 1 Perform a Z rotation (phase), by the specified angle.
Important note: If more than one qubit is specified, this will perform a conditional phase shift, using all specified qubits (like a CZ gate).
rotatex theta_degrees, target_qubits rotatex 315 0 Rotate in X rotation
rotatey theta_degrees, target_qubits rotatey 315 3 Rotate in Y rotation
